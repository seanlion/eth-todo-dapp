## Basic TODO App powered by Ethereum Rinkeby network

### Prerequisite
- Node.js
- npm
- truffle suite
- ganache(local development)
- truffle-plugin-verify
- lite-server(local development)

### Test
- `truffle test`

### Deploy
- `truffle migrate --network rinkeby`

### Verify
- `truffle run verify TodoList --network rinkeby`

### Local App build
- `npm run dev`
